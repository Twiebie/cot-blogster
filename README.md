#Blogster
A responsive theme for Cotonti Siena, built with Twitter Bootstrap.

##Installation:

1. Place the folder 'blogster' in your Cotonti themes folder.
2. Enable the theme by setting: `$cfg['defaulttheme'] = 'blogster';` in datas/config.php.
